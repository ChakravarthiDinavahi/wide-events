# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-20

### Added
- Initial release of wide-events gem
- Rails middleware for automatic event capture
- EventBuilder class for constructing comprehensive events
- Tail sampling implementation
- Controller helpers for enriching events with business context
- Configuration system for sampling rates and rules
- Automatic user context extraction
- Error capture and logging
- Support for high-cardinality, high-dimensionality data
